Welcome to My Portfolio

I arrived at college with the same curiosity most engineers have - eager to understand how the world works. While studying mechanical systems, I stumbled into a programming workshop almost by accident. What started as "just another elective" quickly became an obsession.

Late nights debugging my first Python scripts revealed a new kind of magic - creating functional solutions from pure logic. I found myself voluntarily taking on extra coding projects, from simple calculator apps to web scrapers that organized course materials automatically.

The moment it clicked? Building a weather app that actually worked. Seeing JavaScript fetch real API data and display it with clean React components gave me that "I built this!" rush no textbook ever could.

Now I thrive in that space between ideas and execution, whether crafting full-stack apps with Node.js or solving algorithmic puzzles in C++. When I'm not coding, you'll find me analyzing basketball strategies (my first love) or hunting for the perfect playlist to code to.

Varanasi Metro Route Planner

  • Enables users to choose starting and ending stations
  • Dynamically shows the shortest path with visualization
  • Interactive web UI with real-time user interaction
  • Uses adjacency list to represent metro network efficiently
